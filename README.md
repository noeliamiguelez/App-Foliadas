## Descripcion del proyecto
FOLIADAS API es una aplicacion backend para gestionar unformación sobre foliadas y grupos. Permite:
- Listar foliadas
- Gestionar usuarios y favoritos
- Integrarse con la aplicacion frontend para mostrar informacion
- Añadir, modificar o borrar datos desde postman

## Descripcion del proyecto
Actualmente el proyecto se encuentra en desarrollo, pero los proximos pasos son:

- Implementar autentificacion de usuarios para login
- Integracion con el frontend

## Características de la aplicación 
- CRUD de foliadas y grupos
- Gestión de favoritos por usuario
- Busqueda de foliadas por provincia
